# HoldMate - Smart Locker Booking System

Secure, on-demand locker booking app with email OTP verification and multi-language support.

## Features
- 🔐 Email OTP authentication
- 🌐 Tamil & English language support
- 📍 Find nearby lockers
- 💳 Multiple payment methods
- 📱 QR code access
- 📜 Booking history

## Technologies
- Flutter
- Dart
- EmailJS for OTP
- SharedPreferences
